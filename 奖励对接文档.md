# 社区奖励 API 接口文档


## 概述


社区奖励系统使用链下 ACC 算法计算用户待领取奖励，通过签名机制安全领取。



## 接口列表


### 1. 查询用户社区奖励


获取用户的社区奖励信息（包含所有等级的累积奖励）。


**请求**


```
GET /api/reward/community/{address}/snapshot
```


**路径参数**


| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| address | string | 是 | 用户钱包地址 |


**响应**


```json
{
  "success": true,
  "error": null,
  "address": "0x123...",
  "team_level": 4,
  "small_area_performance": "150000000000000000000000",
  "is_eligible": true,
  "pending_reward": "25000000000000000000"
}
```


**响应字段说明**


| 字段 | 类型 | 说明 |
|------|------|------|
| success | bool | 请求是否成功 |
| error | string? | 错误信息 |
| address | string | 用户地址 |
| team_level | number | 用户当前等级 (0-6) |
| small_area_performance | string | 小区业绩 (wei) |
| is_eligible | bool | 是否符合领取条件 |
| pending_reward | string | 待领取奖励 (wei) |

### 4. 请求领取签名 ⭐


核心接口：验证用户身份后生成服务端签名，用于链上领取。


**请求**


```
POST /api/reward/community/sign
```


**Query 参数**


| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| contract | string | 否 | 合约地址（默认从环境变量读取）|


**请求体**


```json
{
  "user": "0x123...",
  "amount": "1000000000000000000",
  "user_signature": "0x...",
  "message": "CommunityReward Claim Request\n\nAddress: 0x123...\nTimestamp: 1706234567\nChain: 56"
}
```


**请求体字段说明**


| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| user | string | 是 | 用户钱包地址 |
| amount | string | 否 | 请求领取金额 (wei)，不传则领取全部 |
| user_signature | string | 是 | 用户对 message 的签名 |
| message | string | 是 | 签名原文（包含时间戳） |


**签名消息格式**


```
CommunityReward Claim Request


Address: {用户地址}
Timestamp: {Unix 时间戳（秒）}
Chain: {链 ID}
```


**前端生成签名示例**


```javascript
const message = `CommunityReward Claim Request\n\nAddress: ${address}\nTimestamp: ${Math.floor(Date.now() / 1000)}\nChain: 61900`;
const userSignature = await signer.signMessage(message);
```


**响应**


```json
{
  "success": true,
  "error": null,
  "user": "0x123...",
  "amount": "1000000000000000000",
  "nonce": 3,
  "deadline": 1706238167,
  "signature": "0x...",
  "contract": "0xCommunityReward..."
}
```


**响应字段说明**


| 字段 | 类型 | 说明 |
|------|------|------|
| success | bool | 请求是否成功 |
| error | string? | 错误信息 |
| user | string | 用户地址 |
| amount | string | 可领取金额 (wei) |
| nonce | number | 当前 nonce |
| deadline | number | 签名过期时间 (Unix 时间戳) |
| signature | string | 服务端签名 |
| contract | string | 合约地址 |


**错误码**


| 错误信息 | 说明 |
|---------|------|
| 用户签名验证失败 | 用户签名无效或地址不匹配 |
| 签名已过期（超过5分钟）| 请重新签名 |
| 用户不存在 | 未注册的地址 |
| 用户等级不足 | 需要 S3 或以上 |
| 小区业绩不足 | 未达到等级门槛 |
| 当前没有可领取的奖励 | 奖励池为空或已领取 |
| 请求金额超过应得奖励 | amount 参数过大 |

## 前端集成示例


### 完整领取流程


```typescript
import { ethers } from 'ethers';


const CHAIN_ID = 61900; // Mova 链


const COMMUNITY_REWARD_ABI = [
  'function claimWithSignature(uint8 level, uint256 amount, uint256 nonce, uint256 deadline, bytes signature)',
  'function getUserNonce(address user) view returns (uint256)'
];


async function claimCommunityReward(signer: ethers.Signer, apiBaseUrl: string) {
  const address = await signer.getAddress();
 
  // 1. 查询待领取奖励
  const infoRes = await fetch(`${apiBaseUrl}/api/reward/community/${address}/snapshot`);
  const info = await infoRes.json();
 
  if (!info.success || !info.is_eligible) {
    throw new Error(info.error || '不符合领取条件');
  }
 
  console.log(`待领取: ${ethers.formatEther(info.pending_reward)} USDT`);
 
  // 2. 生成用户签名（证明身份）
  const timestamp = Math.floor(Date.now() / 1000);
  const message = `CommunityReward Claim Request\n\nAddress: ${address}\nTimestamp: ${timestamp}\nChain: ${CHAIN_ID}`;
  const userSignature = await signer.signMessage(message);
 
  // 3. 请求服务端签名
  const signRes = await fetch(`${apiBaseUrl}/api/reward/community/sign`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      user: address,
      amount: null, // 领取全部
      user_signature: userSignature,
      message: message
    })
  });
  const signData = await signRes.json();
 
  if (!signData.success) {
    throw new Error(signData.error);
  }
 
  // 4. 调用合约领取
  const contract = new ethers.Contract(
    signData.contract,
    COMMUNITY_REWARD_ABI,
    signer
  );
 
  const tx = await contract.claimWithSignature(
    info.team_level,           // 等级 (3=S3, 4=S4, 5=S5, 6=S6)
    signData.amount,           // 金额
    signData.nonce,            // nonce
    signData.deadline,         // 过期时间
    signData.signature         // 服务端签名
  );
 
  console.log('交易已发送:', tx.hash);
  await tx.wait();
  console.log('领取成功!');
 
  return tx.hash;
}
```


### React Hook 示例


```typescript
import { useState, useCallback } from 'react';
import { useAccount, useSignMessage } from 'wagmi';


const CHAIN_ID = 61900; // Mova 链


export function useCommunityReward() {
  const { address } = useAccount();
  const { signMessageAsync } = useSignMessage();
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);
 
  const claim = useCallback(async () => {
    if (!address) return;
   
    setLoading(true);
    setError(null);
   
    try {
      // 生成签名消息
      const timestamp = Math.floor(Date.now() / 1000);
      const message = `CommunityReward Claim Request\n\nAddress: ${address}\nTimestamp: ${timestamp}\nChain: ${CHAIN_ID}`;
     
      // 用户签名
      const userSig = await signMessageAsync({ message });
     
      // 请求服务端签名
      const res = await fetch('/api/reward/community/sign', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          user: address,
          user_signature: userSig,
          message
        })
      });
     
      const data = await res.json();
      if (!data.success) {
        throw new Error(data.error);
      }
     
      return data;
    } catch (e: any) {
      setError(e.message);
      throw e;
    } finally {
      setLoading(false);
    }
  }, [address, signMessageAsync]);
 
  return { claim, loading, error };
}
```

## 安全说明


1. **用户签名验证**：领取前需要用户签名证明钱包所有权
2. **时间戳验证**：签名消息有效期 5 分钟
3. **Nonce 防重放**：每次领取后 nonce 自增
4. **Deadline 限制**：服务端签名有效期 1 小时
5. **金额限制**：无法领取超过应得奖励的金额
6. **链上最终验证**：合约会验证签名、nonce、deadline 和余额


---




## 升降级奖励说明


- **升级**：原等级奖励被锁定，可继续领取；新等级从升级时刻开始累积
- **降级**：原等级奖励被锁定，可继续领取；新等级从降级时刻开始累积
- **总奖励**：所有曾经达到过的等级奖励累加
